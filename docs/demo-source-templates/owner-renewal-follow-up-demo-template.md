# Owner Renewal Follow-Up Demo Template

Source status: Approved demo seed
Space: Lease Renewals
Audience: Owner
Channel: Gmail
Sensitivity: Low

This is safe demo content for live Ask retrieval. It is not a real owner email and
does not contain client data.

## When To Use

Use this template when the team needs an owner-facing renewal follow-up and the current
renewal details are documented in an approved source.

Do not use this template to approve renewal terms, confirm fees, commit to a timeline,
or promise tenant-facing action unless those facts are present in the cited source.

## Draft Template

Draft - Review before sending

Hi [Owner Name],

We are checking the renewal status and will confirm the recommended next step once the
documented renewal details are verified.

If there is a renewal decision needed from you, we will identify the specific item and
confirm it before any tenant-facing commitment is sent.

Thank you,

## Placeholder Triggers

Create a placeholder instead of a confident answer when any of these are missing:

- exact renewal follow-up timing;
- fee, rent, notice, or term detail;
- owner approval or owner exception handling;
- source link to the lease, SOP, transcript, or approved template.

## Good Ask Questions For Smoke Testing

- What owner renewal follow-up should I send?
- What should the draft say if renewal details are not verified?
- When should a lease renewal answer create a placeholder?
