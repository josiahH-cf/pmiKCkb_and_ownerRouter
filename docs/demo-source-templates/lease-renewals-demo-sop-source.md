# Lease Renewals Demo SOP Source

Source status: Approved demo seed
Space: Lease Renewals
Sensitivity: Low

This is safe demo content for proving the PMI KC KB live Ask path. It contains no real
owner, tenant, address, lease, ledger, bank, screening, or personally identifying data.

## Workflow

1. Check the current lease and renewal status before giving renewal guidance.
2. Confirm owner direction before sending owner-facing commitments.
3. Use approved renewal follow-up wording only when the source is verified.
4. Create a placeholder when timing, fee, approval, or exception details are not
   documented.

## Handling Rules

- Do not invent a renewal deadline, fee, notice rule, owner decision, or tenant-facing
  commitment.
- If the source does not confirm a fact, mark the fact as needing verification.
- If the question asks for an owner-facing reply, produce draft wording for copy/paste
  review only.
- The KB does not send email or update RentVine, LeadSimple, DotLoop, QuickBooks,
  Google Sheets, or any system of record.
