R04 · External Vendor identity, mailbox provider, and account security

App login and mailbox authorization are separate. The app can issue a login, but it cannot read/send as an arbitrary external mailbox without that mailbox owner's provider authorization. Vendor-owned mailbox OAuth also requires a narrow governance exception: all PMI KC cloud/admin/runtime identities stay pmikcmetro.com, while a Vendor token may authorize only that Vendor's own mailbox and assigned tickets.
