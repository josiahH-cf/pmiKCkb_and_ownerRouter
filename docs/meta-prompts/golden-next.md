# Golden Next-Step Feature Set

The single best move after governance: the lowest-risk, highest-leverage, ready-now bundle — no
system-of-record writes, no new vendor risk, and it de-risks everything downstream. It is
decision-complete and readiness-improving, and respects the stop-and-reset rules (it adds no gated
external action and invents no product decision).

1. **Land S1 (governance spine + gate + truncate).** Decision-complete, enforced, unblocks all specs.
   _(Done this cycle: `docs/facts.md`, `npm run verify:context-freshness`, slimmed `docs/loop-state.md`.)_
2. **Run S2 (Voice & Copy).** Read-only string changes grounded in `docs/voice-and-audience.md`; ships
   professionalism immediately; deletes future-promise / over-claim copy. Lowest risk, visible value.
3. **Stand up S9 (local-model provider seam + live-data harness).** Free, fences from prod, de-risks
   every later model-touching feature; keeps the budget guard green.
4. **Start S3 discovery only (golden data + column meanings + process truth).** No build; produces the
   validated facts that gate the renewal work and feed the cross-product glue.

Maintenance intake (S4) and the cross-product glue (S7) follow once S3's facts and the
storage/write-back choices (`Q-WRITEBACK-METHOD`, `Q-MAINT-STORAGE` in `docs/facts.md`) are
owner-approved.
