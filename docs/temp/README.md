# Temporary Planning Artifacts

Use this folder for disposable planning artifacts created during autonomous feature
cycles, including cycle packets, draft client communications, scratch implementation
maps, and temporary meta-prompts.

Rules:

- Do not store secrets, tokens, raw Gmail content, customer records, leases, ledgers,
  bank data, SSNs, or client-private source material here.
- Treat files here as scratch unless the user explicitly asks for a permanent artifact.
- Promote durable decisions into active docs such as `docs/status.md`,
  `docs/client-checklist.md`, `docs/research-backlog.md`, `docs/products/`, or
  `docs/plan.md`.
- Use `docs/autonomous-agent-runner.md` as the active runner before creating new packet
  files.
