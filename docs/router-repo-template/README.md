# Owner Router Repo Template

Use this as a seed for the separate `pmi-kc-owner-router` repository.

This repository should contain configuration and content templates only:

- Six canonical Drive file templates.
- Owner Router Gem system prompt and fallback prompt pack.
- Optional scoped Apps Script setup helpers.
- Gmail filter export placeholder.
- Historical-thread dry-run records.
- Acceptance checklist.

It must not contain a standalone app, a custom database, a KB API client, autonomous
send capability, or writes to external systems.
