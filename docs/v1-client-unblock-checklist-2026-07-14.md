# Final-V1 external unblock checklist

Date: 2026-07-14. Status: **external evidence required; keep every listed production gate closed**.

This is the smallest client/owner checklist left after exhausting repository context and completing
the safe local/emulator work. It is not a blanket approval request. Each row is a separate gate, and
every new/not-yet-proven promotion remains closed until its own evidence, code review, bounded proof,
monitoring, correction path, and explicit authority are recorded. Reused S19 linked reply/label
transports retain only their existing workflow/context/artifact/exact-confirmation authority; their
current Registry state neither proves nor authorizes S25/S26 initiation.

`example.invalid` people/mailboxes plus invented ticket, lease, unit, Vendor, thread, process, folder,
work-order, account, and document references. The typed S22/S25/S26 journey uses these aliases, exact

aliases for development and emulator regression. Never copy them into production mappings or treat a
green synthetic run as provider proof.

## What is not a blocker

- No R01–R09 product decision needs to be asked again.
- The invented aliases, typed adapters, exact preview schemas, one-attempt kernel, S20 queue bridge,
  bounded S21 content store, S24 emulator worker, and pre-V1 manifest/report are implemented locally.
- Generic inbox browsing, free-form compose/send, autonomous/scheduled/bulk/model-triggered sends,
  QuickBooks post/pay, photo replacement/deletion, and guessed provider endpoints are outside authority,
  not items to “unblock.”

## How to close a row

For one row at a time, attach the non-secret contract/mapping evidence, name the credential owner and
secret location (never the value), run the focused synthetic/staging proof in an explicitly permitted
environment, capture only hashes/IDs/status/timestamps, demonstrate monitor and correction/rollback,
obtain exact per-action authority, and then code-review the Action Registry promotion. A failed or
ambiguous proof leaves the row closed and requires reconciliation before another attempt.
